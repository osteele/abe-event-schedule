# Abvent

Display an event schedule from ABE.

## Develop

### Install

1. Install [yarn](https://yarnpkg.com).
2. Install npm and elm packages:

    ```bash
    $ yarn install
    ```

### Run

Run the ABE back end. Or, set the `API_SERVER` environment variable to an ABE
API server.

```bash
$ yarn start
```

### Test (TODO)
